Projeto de Pesquisa - Dashboard para Análise de Sinais Vitais

Propõe o desenvolvimento de um sistema de monitoramento dos sinais vitais de pacientes que requerem atenção médica, fazendo uso de ferramentas da Internet das Coisas, mais especificamente com sensores de batimentos cardíacos e oxigenação do sangue, como modo de promover o acompanhamento em tempo real por parte do profissional de saúde através de um dashboard que mostra os dados relevantes para a análise, estando o mesmo disponível em uma instância na nuvem para que o acesso possa ser realizado remotamente via internet.

Tecnologias utilizadas: Iot, Arduino + Sensores, Node-RED, MQTT, Google Cloud Plataform;
